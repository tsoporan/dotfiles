Setting up vIM:
===============


```
  git clone https://github.com/tsoporan/dotfiles.git dotfiles
```

```
  ln -s dotfiles/.vim/.vimrc ~/.vimrc
```

```
  mkdir -p ~/.vim/bundle/
```

```
git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

```
  vim
```

```
  :PluginInstall
```
